# Images Directory Structure

This directory contains all images used in the Retrofusion website.

## Directory Structure

```
images/
├── hero/           # Hero section background images
├── villas/         # Villa images (retro-villa, neo-retro, retro-viswa)
├── gallery/        # Gallery preview images
├── founders/       # Founders/team photos
└── blog/          # Blog post featured images
```

## Usage

- Place images in the appropriate subdirectory
- Reference them in components using: `/images/[category]/[filename]`
- Use descriptive filenames (e.g., `retro-villa-exterior.jpg`)
- Optimize images for web use (WebP format recommended)

## Image Guidelines

- **Hero images**: 1920x1080px or higher
- **Villa images**: 800x600px 
- **Gallery images**: 400x400px (square)
- **Blog images**: 400x300px
- **Founder photos**: 600x600px
